My past work:

Daky In Monsterland

Daky in Monsterland was my capstone project for the web development immersive I went through at the Galvanize campus in Boulder, CO. It's a reimagining of a game from the mid 80's called Daleks, which I played on the original Mac as a kid. It was a two week solo project, which included the stipulation that it involve at least one unfamiliar technology- that is, one technology that I had never used before. My unfamiliar tech was Phaser, which is a game development engine. I also used animate.css, skeleton.css, and jQuery in building the game, and it features the art of Becky S. Chapin.

Ship, Captain, and Crew

Ship, Captain, and Crew was my first quarter solo project at Galvanize (see above). Like the capstone project, this project also had a special requirement: it had to use an external API call. Ship, Captain, and Crew is pirate-themed version of the dice game of the same name, and it includes an English to Pirate translator. The game was made in one week, using the Materialize framework and jQuery.

Pick My Lunch

Pick My Lunch is a seat-of-my-pants app I made for some folks who often have lunch together, but who can never decide what they want. For now, it's very simple- you type in a list of choices, and it picks one for you at random. I hope to improve it at some point to use an API call to find nearby places to eat, and pick from that list, saving the users even more time.

Handy Weather

Handy Weather was created as a code challenge for a developer job I applied for. While I unfortunately didn't get the job, it was a really good exercise in debugging and trouble-shooting, and a good reminder of how important the fundamentals of code are.
